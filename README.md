# BalloonTec - Official Website

BalloonTec es un equipo independiente de desarrollo de software en Bolivia, enfocado en crear soluciones ágiles y accesibles para resolver problemas reales de la comunidad.

## 🚀 Tecnologías

Este proyecto está construido con un stack moderno y enfocado en el rendimiento y SEO:

- **Framework:** React 19 + ESLint
- **Build Tool:** Vite
- **Styling:** Tailwind CSS v4
- **Routing:** React Router v7
- **Animaciones:** Framer Motion
- **Formularios:** React Hook Form + Yup
- **SEO:** React Helmet Async

## 📦 Desarrollo Local

Clona el repositorio e instala las dependencias:

```bash
npm install
```

Inicia el entorno de desarrollo local:

```bash
npm run dev
```

## 🏗️ Compilación para Producción

Para generar el build estático optimizado para despliegue (Vercel, Netlify, etc.):

```bash
npm run build
```
