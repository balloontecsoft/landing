# BalloonTec Design System

## 1. Theme

Modern Retro-Tech

## 2. Core Principles

- Clean, minimalist, elegant, extremely serious
- Abundant negative space
- High contrast, monochrome base

## 3. Typography

- Primary: Inter or Roboto
- Style: Geometric, sans-serif

## 4. Color Palette

- Base: White (#ffffff), Black (#000000)
- Accents: Light grays (#f3f4f6, #e5e7eb)
- Primary CTA: Emerald Green (#10B981) - ONLY used for Call to Action buttons.

## 6. Design System Notes for Stitch Generation

**Design Aesthetic:** "Modern Retro-Tech". Clean, minimal, elegant, and serious. Use lots of negative space.
**Colors:** Strictly monochrome (white backgrounds, black text, light gray borders or secondary sections). Use a single vibrant accent color (e.g. emerald green) ONLY for primary Call To Action buttons.
**Typography:** Inter or Roboto font face.
